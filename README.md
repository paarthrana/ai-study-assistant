# AI Study Assistant 🤖

A simple AI-powered study assistant built using Python and the Gemini API.

## Features

* Ask academic questions
* Receive AI-generated responses
* Interactive terminal-based interface
* Powered by Google's Gemini model

## Technologies Used

* Python
* Gemini API
* google-genai
* python-dotenv

## Project Structure

ai-study-assistant/

├── main.py

├── README.md

├── notes.md

└── .gitignore

## How It Works

1. User enters a question.
2. Python sends the question to Gemini.
3. Gemini generates a response.
4. The response is displayed to the user.
