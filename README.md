# AI Study Assistant 🤖

A simple AI-powered study assistant built using Python and the Gemini API.

## Features

- Ask academic questions
- Receive AI-generated responses
- Interactive terminal-based interface
- Powered by Google's Gemini model
- Explain Like I'm 5
- Quiz Generation
- Topic Summarization
- Chat History Saving

## Technologies Used

* Python
* Gemini API
* google-genai
* python-dotenv

## Project Structure

ai-study-assistant/
│
├── main.py
├── history.txt
├── .gitignore
└── README.md


## How It Works

1. User selects a study mode.
2. Python sends a prompt to Gemini API.
3. Gemini generates a response based on the selected mode.
4. The response is displayed in the terminal.
5. The conversation is saved to chat history.
